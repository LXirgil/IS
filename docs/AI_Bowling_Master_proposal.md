# AI Bowling Master — 企画書（案）

## 概要
「AI Bowling Master (仮)」は、家庭・リーグ・個人練習向けに投球データを記録・可視化し、AI（Gemini）によるコーチングを提供するモバイルアプリです。

## 主な機能
- 投球データ記録: 各投球のピン倒数、残ピン、動画パス、タイムスタンプを保存
- 統計表示: アベレージ、ストライク率、スペア率、スコア推移の折れ線グラフ
- 苦手ピン分析: 残りやすいピンを自動特定し頻度順で表示
- AIコーチング: Gemini API を用いたスコア分析コメント、残ピン攻略ナビ、動画解析によるフォーム診断

## 技術スタック
- フロントエンド: Flutter
- チャート: `fl_chart`
- 永続化: `Hive` または `sqflite`（要検討）
- AI: Google Gemini（テキスト／画像／ビデオ解析）

## マイルストーン（短期）
1. データモデル実装（完了）
2. 永続化と投球入力画面
3. 統計計算とチャート表示
4. 苦手ピン自動特定
5. Gemini連携の雛形（API呼び出し＋結果表示）
