AI Bowling Master - 企画書(.docx)自動生成

このスクリプトは `python-docx` を利用してワークスペース内に企画書を生成します。

依存パッケージ:
- python-docx

インストール例 (PowerShell):

```powershell
python -m pip install --upgrade pip
pip install python-docx
```

生成コマンド:

```powershell
cd C:\Users\14092\IS
python .\scripts\generate_proposal.py
```

出力: `proposals/AI_Bowling_Master_Proposal.docx`

補足: 生成後は Word で開いて体裁を微調整してください。画像や図を追加する場合は `proposals/` フォルダに画像を置き、スクリプトを拡張してください。
