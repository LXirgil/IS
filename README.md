# AI Bowling Master (仮)

ボウリングのスコア分析とAIコーチングを行うFlutterアプリのリポジトリです。

目的:
- 投球データの記録・可視化（平均、ストライク率、スペア率、ピン別残り傾向）
- 苦手ピンの自動特定と攻略ナビ
- Gemini API を利用したAIコーチング（テキスト・図解・動画解析）

開発メモ:
- データモデル: `lib/models/bowling.dart`
- チャート表示には `fl_chart` を利用予定
- 永続化: Hive / sqflite 検討中
